An exploration of the HTML <ruby> annotation tag as a an interface element. This component library comprises 25 instances of this interface element, either visually or literally.

This tag is essentially used to display small, annotative text—known as "ruby characters"—above or alongside base text, primarily to show pronunciation or meaning.

This website aims to explore the potential of a single HTML element as a versatile tool for communication and interaction design, and to create a cohesive and engaging user experience that showcases the power of this element in different contexts and applications.
